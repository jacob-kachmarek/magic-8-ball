# Html

    -<h1>
    - <section>
        -<p> prompt
        -<button> submit button
        -<input> user input

    -<section>
        -<img> 8 ball image
        -<p> answer display
        -<button> Re-Roll

# Events

    -button click(<submit>)
        -hide prompt
        -display <img>
        -pick random answer from array
        -set content for answer <p> to the   random choice from array

    -button click(Re-Roll)
        -hide 8 ball <img> and <p> answer
        -show prompt
